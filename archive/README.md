# Archive

These files are **not required** to run the BrightHR & BLIP Dashboard. They are kept for reference, historical analysis, or Power BI.

| Contents | Description |
|----------|-------------|
| **CSVs** | Raw/alternate absence data, Power BI–ready exports, org mapping tables |
| **Excel** | BrightHR absence report (Nov–Dec) |
| **Power BI** | `Blip.pbix`, `HRM.pbix` – legacy dashboards |
| **Notebooks** | Analysis, cleaning, and Power BI cleanup (Jupyter) |
| **blip_integration/BlipAppNew.py** | Standalone BLIP script; logic is now in `app.py` |

To run the dashboard you only need, in the project root:

- `AbsenseReport_Cleaned_Final.csv` (Absence tab default)
- `blip_integration/Blip_27_28.xlsx` (BLIP tab default)
